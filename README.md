# AI-Powered Sales Forecasting Dashboard

This project is developed as part of the Future Interns Machine Learning Internship – Task 1.

## Objective
To analyze historical retail sales data and predict future sales trends using machine learning.

## Tools & Technologies
- Python
- Pandas, NumPy
- Scikit-learn (Linear Regression)
- Streamlit
- Plotly

## Dataset
Retail sales data (monthly).

## Features
- Interactive filters (State, Category)
- Sales KPIs
- Past sales trend visualization
- Future sales forecasting

## Output
A scrollable interactive dashboard displaying sales trends and predictions.
